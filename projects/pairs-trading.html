Real-Time Pairs Trading Strategy

A mean-reversion trading system built around cointegration, ADF testing, z-score thresholds, bootstrap uncertainty, and backtesting.

Problem

Pairs trading depends on finding two assets whose spread behaves like it wants to come back home. This project models that spread, tests for stationarity, and builds entry/exit rules around standardized deviations from equilibrium.

Statistical Frame

  • Cointegration to identify pairs with a stable long-run relationship.
  • ADF tests to evaluate spread stationarity.
  • Z-score thresholds for signal generation.
  • Half-life estimation to understand mean-reversion speed.

Uncertainty

  • Parametric and non-parametric bootstrap resampling.
  • Confidence intervals for half-life and expected spread return.
  • Interval-based caution around volatile periods to reduce false positives.

Result

The strategy was backtested in Python with optimized entry and exit rules, producing stronger risk-adjusted performance and a reported Sharpe ratio above 1.6 in the resume version of the project.

Stack

Python, pandas, NumPy, statsmodels, ADF tests, cointegration, bootstrap resampling, backtesting.
