Watch Website Data Analysis

An R Shiny dashboard using regression, clustering, text mining, and visualization to analyze 750+ watch listings.

Problem

Watch prices are driven by brand, model, movement, condition, metadata, and scarcity. This project turns a curated collection of listings into an interactive dashboard for exploring value drivers and market patterns.

Analysis

  • Regression to study price determinants.
  • Clustering to group similar listings or model families.
  • Text mining on model names and metadata.
  • Interactive visual exploration through R Shiny.

Interface

The dashboard is built for exploration, not just static reporting. Users can filter, compare, and inspect patterns across brand, model, movement, and price behavior.

What it shows

This project shows my interest in building data products: not just running models, but creating tools that make the model outputs understandable and useful.

Stack

R, Shiny, ggplot2, regression, clustering, text mining, exploratory data analysis.
