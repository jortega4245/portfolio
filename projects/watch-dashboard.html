Watch Website Data Analysis

An interactive analytics project that treats watch listings like a small market: brands, references, movements, price structure, and collectible patterns.

GitHub Profile

Problem

Watch listings contain a messy mix of brand names, model references, movement details, pricing, and subjective value. The goal was to make that market easier to explore through structured analysis.

Method

The dashboard uses R Shiny to analyze 750+ listings. Regression supports price determinant analysis, clustering groups similar listings, and text mining helps organize watch model names and metadata into more useful categories.

Interface

The project emphasizes interaction: filtering, comparing brands, studying movement types, and making visual patterns easier to see. This makes the project useful both as a data analysis tool and as a product-style portfolio piece.

What This Shows

This project demonstrates dashboard development, applied regression, clustering, text mining, visualization, and the ability to convert a personal domain interest into a structured analytics product.
