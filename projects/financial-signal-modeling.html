Financial Signal Modeling

Sentiment, spectral analysis, periodograms, autoregressive structure, and LLM-generated interpretation for equity market signals.

Problem

Financial markets are noisy, but not always structureless. This project asks whether sentiment and price behavior contain useful temporal patterns, especially short-horizon dependencies and cyclical structure that can be explored through spectral methods.

Data + Features

  • Equity price data for names like NVDA, AMD, TSM, INTC, and AVGO.
  • Financial news sentiment features built from text sources.
  • Lagged sentiment, returns, volatility, and engineered time-series predictors.

Methods

  • Periodograms to inspect frequency power and possible cycles.
  • Autoregressive models to measure temporal dependence.
  • Feature engineering for short-horizon predictive modeling.
  • LLM interpretation layer using LoRA / QLoRA-style fine-tuning ideas.

Why it matters

This project connects two things I like: mathematical signal analysis and practical financial interpretation. The goal is not just a prediction score, but a system that can explain what patterns it thinks it sees.

Stack

Python, pandas, NumPy, statsmodels, spectral analysis, AR models, sentiment engineering, LLM interpretation.
